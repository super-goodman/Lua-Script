                                    • 参数

                                      delayTime: 需要延迟运行的时间，时间基准为1ms

                                    • 返回值：

                                      无

                                    --延迟了1000ms = 1s
                                    sleep(1000)

                                  • printd(“string”,num)

                                     

                                    API作用：打印一个整数型的数据，能在debugview 中看到这个数据

                                    • 参数

                                      string: 格式化字符

                                      num: 一个整数型的数据

                                    • 返回值：

                                      无

                                    使用例子：

                                    xxxxxxxxxx
                                    --能在debugView 中看到：
                                    -- return = 123
                                    -- return = 321
                                    printd("数字 = %d",123)
                                    a = 321
                                    printd("数字 = %d",a)
